(it's short for "Remote Automated Irrigation Network")

What is Irrigation Monitoring and Control System?

This web application provides a real-time interface for monitoring and controlling an irrigation system. Designed for efficient water management, it uses sensor data and user-defined thresholds to automate and optimize the irrigation process. Users can interact with the app to monitor current soil moisture levels, adjust thresholds, and control irrigation modes remotely.

Key Features:

  • Real-Time Humidity Monitoring: The app displays the current soil moisture level in real-time, categorized as Dry, Moist, or Wet. It also dynamically adjusts the display color to visually indicate the soil's condition.
  • Threshold Management: Users can define and update thresholds for "Dry Max" and "Moist Max" levels. These thresholds determine how the system interprets sensor data to automate irrigation.
  • Irrigation Modes:
    • Auto: The system automatically starts and stops irrigation based on soil moisture levels and user-defined thresholds.
    • Manual Irrigation: Users can manually control the irrigation process by toggling the "Irrigate" button.
  • Firebase: The app uses Firebase Realtime Database for communication between the sensors, the irrigation system, and all connected users. Any changes, such as mode toggles or threshold updates, are instantly updated for all devices.
  • Responsive Design: The interface is designed to be user-friendly and works well across various devices, including desktops and smartphones.
