What is Irrigation Monitoring and Control System?

This web application provides a real-time interface for monitoring and controlling an irrigation system. Designed for efficient water management, it uses sensor data and user-defined thresholds to automate and optimize the irrigation process. Users can interact with the app to monitor current soil moisture levels, adjust thresholds, and control irrigation modes remotely.

Key Features:

  • Real-Time Humidity Monitoring: Displays the current soil moisture level in real-time, categorized as Dry, Moist, or Wet. The display color dynamically changes to visually indicate soil conditions.
  • Threshold Management: Users can define and update thresholds for "Dry Max" and "Moist Max" levels, which automate irrigation based on sensor data.
  • Irrigation Modes:
    • Auto: Automatically starts and stops irrigation based on soil moisture levels and thresholds.
    • Manual Irrigation: Users can manually control irrigation by toggling the "Irrigate" button.
  • Firebase Integration: Utilizes Firebase Realtime Database for instant updates across all connected devices, ensuring synchronization.
  • Responsive Design: Works seamlessly on desktops, tablets, and smartphones.
